# AgriConnect
An application which enables farmers to sell their crops to retailers at market price
## Features 🚀
- Farmers can list their products 📋
- Retailers can browse and purchase 🛒
- Market price tracking 📈
- Secure payments 💳

## How to Use
1. Create an account 📝
2. List your crops 🌱
3. Connect with retailers 🤝
